# Environment Variables

## Web app

- `NEXT_PUBLIC_APP_URL`
- `DATABASE_URL` (if using Postgres/Supabase)

## Email

- `EMAIL_PROVIDER`
- `RESEND_API_KEY` (if Resend)
- `EMAIL_FROM`

## GoHighLevel

- `GHL_BASE_URL`
- `GHL_API_KEY`
- `GHL_PDF_EMAIL_WORKFLOW_ID` (optional - workflow ID for PDF email delivery)

## Notes

- Self-host installs should always bring their own keys.
- No secrets committed to git.

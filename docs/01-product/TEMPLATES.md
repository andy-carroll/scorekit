# Templates

## Purpose

Define how audit templates are created, versioned, and shipped.

## Template structure (high level)

Each template contains:

- Positioning docs (ICP, JTBD, promise)
- Questions and answer scales
- Scoring model configuration
- Recommendations mapping rules
- Copy for landing and report sections

## Initial templates

- Founder/MD AI clarity scorecard
- Recruitment agency scorecard
- COO / SMB project & admin chaos scorecard

## Versioning

- Templates must be versioned.
- A report must always record the template version used.
